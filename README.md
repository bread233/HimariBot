# HimariBot

## How to start

1. generate project using `poetry install` .
2. run your bot using `poetry run python bot.py` .

## start with docker

Ubuntu:

sudo docker run --name himaribot --privileged=true -p 8181:8181 --network host -d xmb233/himaribot:v1.0
